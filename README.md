# Rhesis
Rhesis is inspire by [Eloquest](https://github.com/sonnyp/Eloquent) a very useful application that already exists. My goals was to write an alternative to that app but instead of written qt and KDE in mind. Well this was the original idea at least. My main goal was/is to learn more about desktop linux application development, so ... it's a hobby project and I will treat is as such. There may be constant updates, none at all or anything in between. 

## Version 1.0 Goals
- [ ] Windows build? not sure on that
- [ ] Alternative distribution methods like AppImage, Snap... 
- [ ] Autostart with you machine
- [ ] Run as a demon to expose the language server for other apps
- [ ] Release on flathub
- [ ] Pipeline base building of the app
- [ ] I18N for at least 2 languages as a base to add more
- [ ] Different colors per recommendation type
- [x] Tarket a flatpak release
- [x] QT/KDE based ui
- [x] Written in rust as much as possible 
- [x] Basic autocompletions using LanguageTool
